# portfolio-dev

Este repositório representa meu retorno gradual ao desenvolvimento de software.
Aqui organizo estudos, exercícios práticos e pequenos scripts, com foco em clareza,
boas práticas e aprendizado contínuo.

## Estrutura
- `fundamentos/` – exercícios de lógica e base
- `automacoes/` – scripts simples para tarefas do dia a dia
- `estudos/` – anotações e conceitos importantes

## Objetivo
Construir consistência, registrar evolução e manter um portfólio honesto e organizado.
